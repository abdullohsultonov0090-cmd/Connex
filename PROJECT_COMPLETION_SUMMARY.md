## Support & Maintenance

### Common Questions

**Q: How do I add a notification?**
A: Use `addNotificationWithPing('Name', 'action', 'desc', 'category', 'time')`

**Q: Can I change the update interval?**
A: Yes, modify the interval value in line 2861 (default 40000ms)

**Q: How do I customize colors?**
A: Edit CSS variables in lines 10-70 or modify gradient values

**Q: Will this work offline?**
A: Yes, but real-time updates won't work without a backend

**Q: How many notifications can I store?**
A: Unlimited, but recommend keeping under 1000 for performance

### Known Limitations

- Real-time updates limited to 40-second interval (by design)
- Notifications stored in browser (no persistence across browsers)
- Image avatars limited to small sizes (<50KB recommended)
- Video preview thumbnails require CORS headers
